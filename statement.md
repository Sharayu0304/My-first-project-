PROBLEM STATEMENT 
=======
This project is based on text based shopping cart system. It allows users to browse predefined products and manage a cart from the command line. 
It has basic e commerce functional needs and operations such as adding, updatig, removing items, viewing totals, and using a web framework or database.

SCOPE OF THE PROJECT
This application is limited to one single user. It covers basic CRUD operations on cart items, price calculations with dicounts. It only requires single input validation. Storange, user authentication, or payment process in not included in this application.

TARGET USERS 
This is a beginner friendly application for Python learners
who wish to understand and work on menu driven programs, lists and disctionaries. 
It can be used to learn how shopping cart features look like and work in the field of E-Commerce. 


HIGH-LEVEL FEATURES:
Display a catalog of clothing products with their ID Numbers, prices and names. 

Adding the products to the cart, handling new entries and increasing/decreasing quantites of new products.

Viewing the cart and finding the overall total bill on that page itself. The application also supports update or delete featueres.

Calculates the total amount and also supports optional discount codes that can be applied on the total bill itelsef. 

You can clear the entire cart and can exit using a simple numeric menu-bar system.
